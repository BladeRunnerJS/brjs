Welcome to Presenter Live Examples! By the time you read this, you will hopefully have read some of the introductory information on our main website about what Presenter is, how it's structured and also seen Presenter's version of the traditional "Hello World" example. There is also a detailed "worked example", that takes you through the process of creating a real-world trade component using Presenter.

Live Examples gives you something quite different; taking you through individual pieces of Presenter's functionality, and letting you experiment with the code in the HTML "View", and the JavaScript "Presentation Model", both of which are in tabs, at the top right of the screen. Every time you alter something, you'll be able to see the result straight away (no "Submit" button required), in the bottom-right "Results" area of the screen.

Most of the examples will provide links to the relevant bits of the API documentation to tell you about the classes and methods you are using.
