In this example a caplin.element.Formatter is used in conjunction with a caplin.element.Renderer to display prices on screen. The following renderer configuration (specified in 'rendererDefinitions.xml') is used to temporarily apply a 'flashing-up' or 'flashing-down' class for 500ms after the price moves: 

<renderer type="live.examples.presenter.flashing-price">
	<control name="caplin.control.basic.TextControl"></control>
	<downstream>
		<transform className="caplin.element.styler.ClassFlashStyler" >
			<attribute name="duration" value="500"/>
			<attribute name="flashing-up" value="flashing-up"/>
			<attribute name="flashing-down" value="flashing-down"/>
		</transform>
	</downstream>
</renderer>

You can then add some styling, using some CSS like this: 

.flashing-up {
	color:#286221;
	background-color:#cdefbd;
}

.flashing-down {
	color:#841819;
	background-color:#feb3aa;
}
