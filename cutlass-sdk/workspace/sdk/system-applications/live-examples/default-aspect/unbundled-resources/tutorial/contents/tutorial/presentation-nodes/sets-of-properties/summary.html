Sometimes, you might want to apply the same setting to the values of several properties at once, such as disabling all input controls, while a request is sent to the server. The PresentationNode.properties() method can be used to locate all properties that match given criteria, recursively. It returns a caplin.presenter.property.Properties object that contains all matching properties. If you then call setValue() on this object, all the returned properties will be updated with the new value.

In this example, all of the fields within the Presentation Model can be enabled and disabled at the same time.
