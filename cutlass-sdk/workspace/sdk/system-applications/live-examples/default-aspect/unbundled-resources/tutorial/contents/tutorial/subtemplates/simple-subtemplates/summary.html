Sub-templates let you split large HTML structures into smaller chunks, making the View easier to understand. A template is simply a top-level HTML tag, such as a <div>, with an id attribute. It's included in the main View template by using a data-bind="template:{}" attribute, which causes the sub-template's contents to be displayed in the element that contains the binding.

You can group multiple sub-templates together in a single file, or split them out into separate files.
