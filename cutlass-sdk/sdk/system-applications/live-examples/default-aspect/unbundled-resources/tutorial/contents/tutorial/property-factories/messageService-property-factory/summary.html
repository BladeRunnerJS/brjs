The caplin.presenter.domain.property.MessageServicePropertyFactory class is used to create properties that are bound to data in the records inside the streaming service provided by the message service.

In the example, two properties are bound to data in records that have /FX/EURUSD as the subject (all of them have static data as we are using a MessageServiceStub to retrieve data):

  • The bestBid property is bound to the BestBid field.
  • The bestAsk property is bound to the BestAsk field.
