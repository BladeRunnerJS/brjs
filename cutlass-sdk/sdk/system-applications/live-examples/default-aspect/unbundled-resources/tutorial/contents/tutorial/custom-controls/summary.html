As well as binding them to HTML form controls, you can bind the Control Models supplied with Presenter, to custom JavaScript controls, such as ExtJs, JQuery, and suchlike. This flexibility gives you the potential to create far richer user interfaces than can be provided with HTML alone.

To allow third-party controls to be used with Presenter, a caplin.presenter.control.ControlAdaptor must be written to provide a standard interface to the control.

Example control adaptors are:

  • ExtJsSelectBoxControlAdaptor
  • ExtJsComboBoxControlAdaptor
  • JQueryDatePickerControlAdaptor
