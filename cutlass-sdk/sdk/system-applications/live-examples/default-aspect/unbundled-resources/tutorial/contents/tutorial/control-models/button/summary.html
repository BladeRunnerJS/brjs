The caplin.presenter.node.Button stores the state of a button and allows properties, such as its visibility, to be set via the model.

Try changing some of the properties in the example below.

You don't have to declare the status of the enable and visible properties in the data-bind statement, if you only want them to keep their default settings; but if you do want to manipulate them using the Presentation Model, the properties must be declared in the View.
