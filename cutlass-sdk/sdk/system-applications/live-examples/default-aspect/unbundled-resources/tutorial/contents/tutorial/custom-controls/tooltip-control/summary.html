This control (referenced though the br.tooltip alias) provides a way to display a tooltip every time there is an error on any of the contained fields.

The tooltip control will use a caplin.presenter.node.ToolTipNode to model it and an caplin.presenter.util.ErrorMonitor class which will keep a stack of the fields that are on error in order to display the tooltip under the appropriate field. Lastly, the fields that are contained within this caplin.presenter.node.ToolTipNode will be modelled as caplin.presenter.node.ToolTipField.
