Normally, you specify which sub-templates are to be used, from data-bind attributes in the main template. However, a PresentationNode may define which template is used to render it, by implementing the caplin.presenter.node.TemplateAware interface. This defines a single method - getTemplateName() - which the PresentationNode can then use to identify the name of a template in the View.

This example creates a node-list containing two legs, but where each leg dynamically specifies its own render template.
