The caplin.presenter.node.MultiSelectionField class is a control model that lets users choose multiple items from a list of options.

Internally, options are stored within a caplin.presenter.node.OptionsNodeList object, which extends NodeList. This allows you to use sub-templates to render each option, while the view is automatically refreshed when the options are updated.

This example shows how a single MultiSelectionField can be rendered concurrently, using both <select multiple="multiple"> and <input type="checkbox" > elements. Additionally, we've added a check-box to show how the list of available options can be dynamically modified. To get multiple options selected initially, we have added the pChosen array to the as a second argument to the MultiSelectionField.
