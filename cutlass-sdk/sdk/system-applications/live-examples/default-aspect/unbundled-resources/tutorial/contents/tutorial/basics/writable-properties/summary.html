The caplin.presenter.property.WritableProperty class extends caplin.presenter.property.Property, adding a setValue(sValue) method that can be called from within the presentation model.

This example updates the amount property after construction.

If you want to be able to use the Presentation Model to dynamically update the value of a property once it's been created, but don't need end users to be able to change it, you would use a Writable Property:

caplin.presenter.property.WritableProperty

A Writable Property is an extension of the basic Property class, that allows you to update its value using the setValue() method.

In this example, we have made the amount property writable, and then used this.amount.setValue(42) to change its value to 42.

The HTML View has not changed at all.
