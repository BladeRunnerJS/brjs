The caplin.presenter.domain.property.SL4BRecordPropertyFactory class is used to create properties that are bound to data in the SL4B records, streamed by Liberator.

In the example, three properties are bound to data in records that have /FX/EURUSD as the subject:

  • The bestBid property is bound to the BestBid field.
  • The bestAsk property is bound to the BestAsk field.
  • The recordAvailable property is bound to an availability flag in the SL4B record, that indicates whether or not Liberator can supply the required data.
