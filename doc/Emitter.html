Class: Emitter

Emitter

Emitter provides event emitting capabilities, similar to Backbone. For more information see the project page.

new Emitter()

This constructor function can be used directly, but most commonly, you will call it from within your own constructor. e.g. Emitter.call(this); It will set up the emitter state if called, but it is optional.
